## Project Overview
The repository includes completed tasks for the Digital Literacy
project. The purpose of this project is to acquire basic digital
literacy skills such as advanced search skills, information
evaluation, professional communication, and cybercrime awareness.

---

## Repository Structure

**task-1-presentation/** — Contains documentation on advanced
Google search operators and effective research techniques.

**task-2-portfolio/** — Includes the analysis and evaluation of
online information for credibility and bias.

**task-3-platforms/** — Documents the use of AI tools as a
collaborative partner in research and drafting.

**task-4-email-etiquette/** — Contains professional email drafts
and a social media safety checklist.

**task-5-cybercrime/** — Features a case study on Phishing and
a 10-point Stay Safe Online prevention checklist.

**report/** — Contains the comprehensive final project report
summarizing all learnings.

---

## Module Summaries

### Task 1 – Digital Literacy Infographic
Created a one-page infographic using Canva covering what
digital literacy is, useful tools, safe internet practices,
professional online presence, and email etiquette.

### Task 2 – Student Digital Portfolio
Set up professional profiles on GitHub, LinkedIn, and Kaggle.
Filled in education details and created a GitHub profile README.

### Task 3 – Coding & Collaboration Platforms
Completed a beginner challenge on HackerRank and created
a 5-question Digital Literacy Awareness Quiz using Google Forms,
linked to Google Sheets for response tracking.

🔗 Digital Literacy Quiz Form: https://forms.gle/MZ9urS2s8mSvXxds5

### Task 4 – Email Etiquette
Drafted two professional emails — one requesting an assignment
extension and one expressing internship interest. Also created
a Social Media Do's and Don'ts checklist.

### Task 5 – Cybercrime Awareness
Researched phishing attacks and wrote a detailed case study.
Created a Stay Safe Online prevention checklist with UPI safety
tips and reporting channels.
